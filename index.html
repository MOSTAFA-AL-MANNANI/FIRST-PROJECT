Parlez d’un projet que vous avez développé

Oui, j’ai développé un projet e-commerce complet avec Laravel :
Ce projet permet à plusieurs boutiques d’avoir chacune un espace personnel pour vendre leurs produits. Chaque boutique dispose d’un lien personnalisé pour consulter uniquement ses propres produits.

Technologie utilisée : framework Laravel (architecture MVC)

Fonctionnalités développées :

  1. Authentification des boutiques
  2. Ajout et gestion des produits
  3. Page boutique publique avec lien unique
  4. Ajout au panier
  5. Envoi de la commande via WhatsApp
  6. Page d’administration pour gérer les boutiques et les commandes

Ce projet m’a permis de renforcer mes compétences en Laravel, de mieux comprendre la logique d’un projet professionnel, de gérer les relations entre modèles, sécuriser les routes et organiser le code proprement.

Parlez de votre projet de fin d’études (PFE)

Pour mon projet de fin d’études, j’ai développé une application web pour une agence de voyage complète en utilisant React pour le frontend et Laravel pour le backend.

Fonctionnalités côté utilisateur :

  1. Parcourir les offres de voyage
  2. Réserver les offres en ligne
  3. Contacter l’agence via un formulaire de contact

Fonctionnalités côté administration :

  1. Gérer les réservations
  2. Gérer les utilisateurs et les messages
  3. CRUD des offres

J’ai utilisé Axios pour la communication entre Laravel et React via une API REST avec JSON. Ce projet m’a permis de renforcer ma maîtrise du développement full stack, la conception d’API, et l’architecture MVC côté Laravel.

Parlez de votre projet de stage PFE

Pendant mon stage chez Oracea Services, j’ai développé une application web en React.js qui permet à un client de :

  • Télécharger un fichier Excel
  • Utiliser un fichier JSON de configuration pour identifier les colonnes et les données importantes
  • Générer automatiquement un fichier XML conforme au format requis

Ce projet est entièrement frontend (sans backend). J’ai utilisé la bibliothèque xlsx pour lire les fichiers Excel, et une logique personnalisée pour créer le fichier XML.
